```python

if a < b:
    for i in range(min(a, b), max(a, b) + 1):
        print(i, end=' ')
```

J. Маршрут построен

```python
ans = [0, 0]
while (s := input()) != 'СТОП':
    cnt = int(input())
    if s == 'СЕВЕР':
        ans[0] += cnt
    elif s == 'ЮГ':
        ans[0] -= cnt
    elif s == 'ВОСТОК':
        ans[1] += cnt
    else:
        ans[1] -= cnt
for i in ans:
    print(i)
```

R. Простая задача 2.0
```python

    while n % i == 0:
        n //= i
        ans.append(i)
print(*ans, sep=' * ')
```

```python

print(m)

while (s := str(input())) != 'Угадал!':
```

T. Хайпанём немножечко!
```python
ans, hn1 = -1, 0
for i in range(int(input())):
    b = int(input())
    h, r, m = b % 256, (b // 256) % 256, b // 256 ** 2
    t = ((m + r + hn1) * 37) % 256
    if t != h or h > 99:
        ans = i
        break
    hn1 = h
print(ans)
```
