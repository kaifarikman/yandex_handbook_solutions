rows, cols = int(input()), int(input())
width = len(str(rows * cols))

for i in range(1, rows + 1):
    line = []
    for j in range(cols):
        if j % 2 == 0:
            num = i + 2 * rows * (j // 2)

P. Редизайн таблицы умножения
```python
size, width = int(input()), int(input())

for i in range(1, size + 1):
    cells = []
    for j in range(1, size + 1):
        value = str(i * j)
        if width % 2:
            value += " "
        cells.append(value.center(width))
    print('|'.join(cells))
    if i < size:
        separator_length = (width + 1) * size - 1
        print('-' * separator_length)
```

T. Математическая выгода
```python
number = int(input())

mx_s = 0
ans = 2

for base in range(2, 11):
    n = number
    digit_sum = 0
    while n > 0:
        digit_sum += n % base
        n //= base

    if digit_sum > mx_s or (digit_sum == mx_s and base < ans):
        mx_s = digit_sum
        ans = base

print(ans)
```
