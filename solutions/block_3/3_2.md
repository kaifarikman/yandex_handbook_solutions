```python
d = {}
while line := str(input()).split():
    a, b = line
    d.setdefault(a, set()).add(b)
    d.setdefault(b, set()).add(a)

for person in sorted(d):
    second_level = set.union(*(d[f] for f in d[person])) - {person} - d[person]
    print(f"{person}: {', '.join(sorted(second_level))}")
```

S. Частная собственность
```python
n = int(input())
d = dict()

for _ in range(n):
    name, toys_str = input().split(': ')
    toys = set(toys_str.split(', '))

    for toy in toys:
        d[toy] = d.get(toy, 0) + 1

unique_toys = [toy for toy, count in d.items() if count == 1]
unique_toys.sort()

for toy in unique_toys:
    print(toy)
```
T. Простая задача 4.0
```python
from math import gcd

numbers = sorted(set(map(int, input().split('; '))))

for a in numbers:
    ans = [b for b in numbers if a != b and gcd(a, b) == 1]
    if ans:
        print(f'{a} - {", ".join(map(str, ans))}')
```
